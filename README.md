# playwright_tests

Playwright tests repo is an example of playwright tests implemented using PageObject pattern and fixtures for learning purposes

## Installation

```bash
npm install
```

## Usage

### Run tests:
```bash
npm run test
```

### Launch tests in UI mode:
```bash
npm run test-ui
```

### Run eslint rules and fix errors if possible:
```bash
npm run lint
```
